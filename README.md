# For launching Flask API (OPTIONAL)

## Create virtual Env:
>>py -m venv stenv

## Launch Virtual Environment:
>> stenv\Scripts\activate

## Install Flask
>> pip install flask

## Set app to default
>> set FLASK_APP=app.py

## Running Flask
>> flask run
>> Ctrl + C to stop
>> deactive [To leave virtual environment]

API Base URL: http://127.0.0.1:5000/api/
